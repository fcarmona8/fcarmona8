Aprendizaje del ciclo (DAW)

Durante el transcurso de los proyectos del segundo año del Ciclo Formativo de Desarrollo de Aplicaciones Web (DAW), he adquirido una amplia gama de habilidades y conocimientos fundamentales en el campo del desarrollo web. Estas experiencias prácticas no solo han fortalecido mi comprensión teórica, sino que también me han proporcionado una base sólida para enfrentar desafíos reales en el mundo laboral.

  • Git:

    Aprendí a utilizar Git como un sistema de control de versiones distribuido, permitiéndome colaborar de manera eficiente en proyectos de desarrollo de software. A través de la creación y gestión de repositorios, así como el uso de ramas y fusiones.


  • JavaScript (JS):

    Con JavaScript, he adquirido habilidades en la manipulación del DOM, la gestión de eventos, y el uso de estructuras de datos y algoritmos para crear funcionalidades avanzadas y experiencias de usuario enriquecidas.


  • HTML y CSS:

    HTML y CSS son los pilares fundamentales del desarrollo web. He aprendido a estructurar y diseñar páginas web utilizando HTML para la marcación semántica y CSS para la estilización y presentación.


  • PHP y Laravel:

    Con PHP, he explorado el desarrollo del lado del servidor, permitiéndome crear aplicaciones web dinámicas y escalables. Además, he trabajado con el framework Laravel, que proporciona herramientas poderosas para simplificar el desarrollo web y seguir las mejores prácticas de arquitectura MVC (Modelo-Vista-Controlador).


  • Figma:

    Figma ha sido una herramienta para el diseño de interfaces de usuario y prototipado interactivo. A través de su plataforma basada en la nube, he podido colaborar eficientemente en el diseño de proyectos web.


  • Trello:

    Trello es una herramienta de gestión de proyectos que he utilizado para organizar tareas, asignar responsabilidades y hacer un seguimiento del progreso en los proyectos de desarrollo web. Su sistema de tableros, listas y tarjetas ha facilitado la planificación y coordinación del trabajo en equipo.



En resumen, los proyectos del segundo año de DAW me han proporcionado una experiencia práctica invaluable, permitiéndome desarrollar habilidades técnicas y competencias profesionales esenciales para mi futura carrera en el campo del desarrollo de aplicaciones web. Estoy entusiasmado por seguir ampliando mis conocimientos y enfrentar nuevos desafíos en este apasionante campo.

Ferran Carmona

Estudiante de desarrollo web
